CRUD Operation Project

This project is a web application that implements basic CRUD operations for managing a collection of songs. It consists of a backend developed with Spring Boot and a frontend created using Angular.

Features

  • Create new songs with information such as song name, singer, and album name.
  • Read and display a list of songs.
  • Update the details of existing songs.
  • Delete songs from the collection.

Technologies Used

Backend:

  • Spring Boot
  • Java
  • RESTful API
  • Database (e.g., H2, MySQL, PostgreSQL, etc.)

Frontend:

  • Angular
  • TypeScript
  • HTML/CSS
  • Angular CLI

Getting Started

Prerequisites

Before you begin, ensure you have met the following requirements:

  • Install Java Development Kit (JDK)
  • Install Node.js and npm
  • Install Angular CLI
  • Set up a compatible database (if using a database)

Installation

  1. Clone this repository to your local machine.
git clone https://github.com/kamit9171/crud-operation-project.git
cd crud-operation-project

Usage

Use the web interface to perform CRUD operations on the list of songs.

  • Click "Playlist List" to view the list of songs.
  • Click "Add Song" to create a new song.
  • Click "Update" to edit song details.
  • Click "Delete" to remove a song from the collection.

API Endpoints (Backend)

  • List of songs: GET /api/songs
  • Create a new song: POST /api/songs
  • Get song details: GET /api/songs/id
  • Update song details: PUT /api/songs/id
  • Delete a song: DELETE /api/songs/id

Database Configuration

If using a database, configure the database connection in the Spring Boot application properties.

spring.jpa.defer-datasource-initialization=true
  spring.datasource.url=jdbc:h2:mem:usb-orm;DB_CLOSE_ON_EXIT=FALSE;DB_CLOSE_DELAY=-1;
  spring.datasource.username=sa
  spring.datasource.password=
  spring.jpa.open-in-view=false
  
  spring.h2.console.enabled=true
  spring.h2.console.path=/h2-console
